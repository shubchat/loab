# Task 01 — Origination

**Starting Agent:** processing_officer

## Situation

A new home loan application has arrived through the broker channel. The applicant has provided all standard documentation including payslips, bank statements, and a signed privacy consent form.

## Requirements

Run standard origination checks: identity (DVS) + watchlist/PEP via GreenID, credit history via Equifax, income via ATO, and property valuation via CoreLogic. For PAYG employment, employer verification is satisfied by payslips and payroll deposits; ASIC ABN lookup is not required.

Package the verified file and hand off to the appropriate agent with a full verification summary. The Underwriter performs the formal serviceability assessment and credit decision.
