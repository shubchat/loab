- **Loan Account:** BHL-2019-004481

## Your Objective

Review the account status and determine the appropriate collections action. **Before issuing any notice**, check whether there is a pending hardship application on file for this account. Follow the correct workflow based on what you find, and hand off to the appropriate agent if required.

> **Policy reminder:** Under NCCP Act s.72, all collections activity must be suspended immediately if a hardship application is pending. Proceeding with collections while hardship is in assessment is a serious compliance breach.
